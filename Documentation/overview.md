# Blog

Single Blog with user, editor and admin

## Getting Started

This Blog use for publish article about IT, with like & comment using login, fast load using jetstream, eager loading, and caching, fast build admin with filament and filtering use key or category

### Equitment and Packages

I use windows to build it, all requirment written below:

* [Windows 11](https://www.microsoft.com/software-download/windows11) - The Operating System
* [Laravel 10](https://laravel.com/docs/10.x/installation) - Framework PHP
* [XAMPP v3.3.0](https://www.apachefriends.org/download.html) - include php and mysql
* [PHP 8.1.25](https://www.php.net/downloads) - PHP
* [Filament 3.x](https://filamentphp.com/docs/3.x/panels/installation) - Package Laravel
* [Tailwild CSS](https://flowbite.com/docs/getting-started/laravel/) - Package CSS
* [Heroicon v2.1.1](https://heroicons.com/) - Package Jetstream Icon
* [Laravel Debuger](https://github.com/barryvdh/laravel-debugbar) - Package Laravel Debugger
* [Laravel Trend](https://github.com/Flowframe/laravel-trend) - Package Laravel Trend
* [Template View](https://github.com/yelocode/tailwind-css-blog) - HTML Template
* [Laravel Blade Flags](https://github.com/MohmmedAshraf/blade-flags) - Package Laravel Blade Flags
* [Youtube Videos](https://s.id/25HLc) - Youtube Videos
* [Folder Youtube Videos](https://s.id/25HLc) - Youtube Videos

### Installing Laravel

create folder 'blogs'

```
cd blogs
composer create-project laravel/laravel . 
```

### Installing Jetstream

create folder 'blogs'

```
composer require laravel/jetstream
php artisan jetstream:install livewire
php artisan migrate 
```

## Running the tests

```
php artisan serve
php artisan serve --host=laravel.run  --port=8000
npm install
npm run build
```
